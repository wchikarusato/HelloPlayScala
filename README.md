Play-Scalaのサンプル

## 実行方法
```
$ cd クローンしたリポジトリ\HelloPlayScala\Application
$ play run
```
## Eclipseにプロジェクト入れる
```
$ cd クローンしたリポジトリ\HelloPlayScala\Application
$ play eclipse

  Eclipse立ち上げる
  → Project Explorerの開いてる場所を右クリック → inport → Inport
  → existing projects でフィルタリング
  → Select root directory に上でplayコマンドした場所を指定してfinish
```
